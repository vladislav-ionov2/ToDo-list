To-Do Liste (einfaches JS Projekt)

Dies ist eine einfache To-Do App, die ich gebaut habe, um die Grundlagen von HTML, CSS und JavaScript zu üben.  
Die Anwendung funktioniert komplett ohne Frameworks und speichert die Aufgaben im `localStorage`, damit sie beim Neuladen nicht verschwinden.

---

Funktionen:

- Neue Aufgaben hinzufügen  
- Aufgaben abhaken (erledigt / nicht erledigt)  
- Aufgaben löschen  
- Zähler für erledigte und gesamte Aufgaben  
- Kleine Animation, wenn alle Aufgaben erledigt sind  
- Aufgaben werden automatisch im Browser gespeichert  
- Einfaches, responsives Design

---

Verwendete Technologien:

- **HTML5**
- **CSS3**
- **JavaScript (ES6)**  
- **localStorage** für das Speichern der Aufgaben

---

Wie startet man das Projekt?

1. Repository herunterladen oder klonen  
2. Datei **index.html** im Browser öffnen  
3. Fertig – die App läuft komplett ohne Server
